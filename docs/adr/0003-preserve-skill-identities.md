# Preserve existing skill identities

We decided to keep `skills/drawio` with `name: drawio` and `skills/drawio-academic-skills` with `name: drawio-academic-skills` while rewriting their descriptions and body structure. This preserves existing install paths and trigger references while allowing the roles to change: `drawio` becomes the shared base capability surface, and `drawio-academic-skills` becomes the publication-focused overlay.
