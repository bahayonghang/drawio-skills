# YAML-first authoring remains the default

The Draw.io Base Skill keeps YAML-first authoring as the default source of truth for create, edit, replicate, import, and conversion workflows. Direct XML generation from the upstream reference remains an exception path for tiny one-off diagrams, CLI-unavailable handoff, or raw mxGraph control; upstream XML guidance is useful as reference material but does not replace the repository's YAML/CLI/sidecar model.
