# 编辑图表

学习如何通过自然语言与基于 ID 的操作来修改现有图表。

## 常见编辑方式

1. **自然语言**：描述你想改什么
2. **基于 ID 的操作**：对特定 cell 做精确修改（更新/新增/删除）

## 自然语言编辑

```
"将'处理'节点的标签改为'验证输入'"
```

## 批量编辑与性能建议

- 尽量把多个修改合并到一次 `edit_diagram` 调用中（批量 operations）
- `get_diagram` 在大图上会返回完整 XML，建议只在需要查 ID 时使用
- 先“粗后细”：先把结构与布局摆正，再统一样式和字体
