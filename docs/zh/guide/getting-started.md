## 什么是 Draw.io 技能？

Draw.io 技能是一个 Claude Code 技能，支持 AI 驱动的图表创建和编辑，提供实时浏览器预览。

## 前置要求

- [Claude Code CLI](https://github.com/anthropics/claude-code)
- [Node.js](https://nodejs.org/)
