The bundled assets/catalog/shape-index.json.gz is derived from the shape index
published by jgraph/drawio-mcp and is used to generate the offline catalog.

Source: https://github.com/jgraph/drawio-mcp
License: Apache License, Version 2.0

This repository stores the unmodified upstream index and a deterministic,
filtered catalog generated by scripts/tools/build-shape-catalog.js.
